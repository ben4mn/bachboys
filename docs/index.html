Features

Everything the crew needs.
Nothing they don't.

Built for the organizer who doesn't want to manage 47 group chat threads. One app, one source of truth.

Event Schedule

Create a full itinerary with times, locations, map links, and per-event costs. Grouped by day, sorted by time.

Cost Splitting

Even splits, custom amounts, or fixed per-person pricing. Automatic recalculation when attendees change. Per-event paid status.

RSVP Tracking

Going, maybe, declined — per event. Mandatory events flagged separately. Avatar stacks show who's in at a glance.

Photo Gallery

Shared photo uploads with image compression. Everyone contributes, everyone sees. No more "send me that pic."

Push Notifications

VAPID-powered web push. Admin can notify all or by event. Works even when the app is closed.

Admin Panel

Full control. Manage events, confirm payments, view balances, send notifications, and toggle user roles.

Screenshots

Dark mode. Light mode.
Always looks good.

Designed mobile-first with Tailwind CSS. Installable as a PWA on iOS and Android.
